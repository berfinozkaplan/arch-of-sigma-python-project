# Python - Görev Yönetim (To-Do List) Uygulaması

Bu proje, Arch of Sigma Bilgi Teknolojileri staj programı kapsamında geliştirilmiştir. Python dili kullanılarak temel programlama mantığının ve veri yapılarının pekiştirilmesi amaçlanmıştır.

### Proje Özellikleri:
*   **Veri Yapıları:** Uygulama içinde görevler listeler (list) kullanılarak dinamik olarak yönetilir.
*   **Koşullu İfadeler:** Kullanıcının seçimine (ekleme, silme, listeleme) göre `if/elif/else` blokları ile mantıksal yönlendirmeler yapılır.
*   **Dosya İşlemleri:** Girilen görevler `gorevler.txt` adlı metin dosyasına kalıcı olarak kaydedilir ve program her açıldığında bu dosyadan okunarak verilerin kaybolması önlenir.
*   **Hata Yönetimi:** Dosya bulunamaması (FileNotFoundError) veya hatalı girdi (ValueError) durumlarında `try/except` blokları ile programın çökmesi engellenir.

### Kullanılan Teknolojiler:
*   Python 3.x
*   Google Colab (Geliştirme Ortamı)
